# Contributing to IRA

Thank you for your interest in IRA (Intelligent Responsive Assistant)!

Please note that this project is primarily developed and maintained solely by **Revant (@rebantsahu)**. 

### Guidelines
- At this time, since the project is personal, major new features are handled directly by the author.
- If you find any bugs or have feature requests, please open an Issue on the repository.
- Pull Requests are welcome for bug fixes, documentation improvements, or minor enhancements. Please open an issue first to discuss your proposed changes before submitting a PR.

Thank you for helping make IRA better!
