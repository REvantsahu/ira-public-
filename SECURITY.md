# Security Policy

## Supported Versions

We actively support and apply security updates to the latest release of IRA.

| Version | Supported          |
| ------- | ------------------ |
| 0.5.x   | :white_check_mark: |
| < 0.5   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do NOT open a public issue. Instead, report it directly to the maintainer:

- **Contact:** Reach out to Revant (@rebantsahu) or via the email/contact details listed on their GitHub profile.
- Please provide a detailed description of the issue, steps to reproduce, and any potential impact.
- We will acknowledge your report and work to address the vulnerability as quickly as possible.
