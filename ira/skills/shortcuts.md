# Skill: Keyboard Shortcuts

This skill tells you how to use common keyboard shortcuts instead of manually clicking buttons for common tasks. This makes your execution much faster and more reliable.

## Browser Shortcuts (Chrome/Edge)
- `Ctrl + T`: Open a new tab
- `Ctrl + W`: Close the current tab
- `Ctrl + Shift + T`: Reopen the last closed tab
- `Ctrl + L` or `Alt + D`: Focus the address bar (to type a URL or search)
- `Ctrl + R` or `F5`: Reload the page
- `Alt + Left Arrow`: Go back
- `Alt + Right Arrow`: Go forward
- `Ctrl + J`: Open Downloads
- `Ctrl + H`: Open History

## Windows System Shortcuts
- `Win + I`: Open Windows Settings
- `Win + E`: Open File Explorer
- `Win + D`: Show Desktop (minimize all windows)
- `Win + L`: Lock the computer
- `Win + S`: Open Windows Search
- `Win + V`: Open Clipboard history
- `Alt + Tab`: Switch between open apps
- `Alt + F4`: Close the current active window
- `Ctrl + Shift + Esc`: Open Task Manager

## Text Editing & General
- `Ctrl + C`: Copy
- `Ctrl + X`: Cut
- `Ctrl + V`: Paste
- `Ctrl + Z`: Undo
- `Ctrl + Y`: Redo
- `Ctrl + A`: Select all
- `Ctrl + F`: Find text on screen

Whenever you can achieve a task using these shortcuts via the `hotkey` tool (e.g. `hotkey(["ctrl", "t"])`), prefer it over trying to find and click the specific buttons!
