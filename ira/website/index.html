◈ Superpowers

88+ Built-in Tools & Vision Systems

From native Python OS controls to multi-key Gemini vision and browser automation, IRA handles it all.

Real-Time Screen Vision

IRA takes silent verification screenshots with MSS and Pillow. She reads buttons, recognizes UI coordinates, and navigates desktop software with human precision.

Native PC Control

Direct mouse clicks, key combinations, app launching, process management, and volume controls via PyAutoGUI and psutil.

Gemini Live Voice Engine

Sub-200ms real-time voice conversations using `gemini-3.1-flash-live-preview`. Speaks Hinglish naturally with full audio barge-in support.

Persistent Memory System

Saves preferences, past conversation context, and user knowledge across restarts. Structured JSON graph queryable by IRA.

Automated WhatsApp & DOM Web

DOM-level browser automation. IRA opens WhatsApp Web, finds contacts, formats text, and sends messages automatically.

Multi-Key Rotation Engine

8-Key Gemini rotation pool + 5-Key Sarvam TTS pool ensuring high throughput with zero rate-limit downtime.
