What is IRA v0.5 Beta? 01

IRA (Intelligent Responsive Assistant) is an autonomous desktop AI agent built using Python 3.10+, Google Gemini APIs, PySide6 QML overlay, and Playwright DOM automation. She runs locally on Windows, reads your screen, controls your mouse & keyboard, and communicates via real-time audio.

How does IRA see and read the computer screen? 02

IRA captures verification screenshots using MSS and Pillow. She extracts UI elements, button coordinates, and text using Gemini 3.5 Flash vision and computer use vision models to execute pixel-precise actions.

What voice engine powers IRA's speech? 03

IRA connects directly to the Gemini Live WebSocket API (`gemini-3.1-flash-live-preview`) for sub-200ms latency voice responses. She speaks Hinglish naturally with full audio barge-in interruption capabilities.

How does IRA automate WhatsApp Web and web browsers? 04

IRA integrates a dedicated WhatsApp Node.js bridge and DOM browser agent. She opens WhatsApp Web, selects contacts, formats text messages, and sends them automatically without manual clicking.

Does IRA remember context and past conversations? 05

Yes, IRA features a persistent JSON memory graph (`memory.py`). She stores user preferences, project details, past tasks, and custom knowledge queryable across application restarts.

Is IRA open source and how can I install it? 06

IRA is 100% open-source under the MIT license created by Revant Sahu @ Nagchetra Labs. You can clone the code and run one-click setup via GitHub at https://github.com/REvantsahu/ira-public-.
