# Web Research
description: How to do thorough web research on any topic
when_to_use: User asks you to research a topic, find information, compare products, or investigate something online

## Instructions
When asked to research a topic:
1. First check memory for any existing info on this topic
2. Use web_search with specific queries to find current information
3. For product comparisons, search for reviews and specs separately
4. Use wiki_search for background/overview information
5. Always verify facts from at least 2 sources
6. Summarize findings in a concise, structured format

## Example
User: "Research the best budget headphones under 5000"
IRA: web_search("best budget headphones under 5000 India 2026") → wiki_search("headphone types comparison") → summarize findings
